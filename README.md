Projeto de automação e hacking com Python, utilizando PyQt5, pyautogui, requests, pyfiglet e bcrypt.

## Funcionalidades

- Automação de tarefas
- Interface gráfica
- Encriptação de senhas
- Ferramentas de hacking ético
